basic-webapp
============

Basic Web app with tests, tomcat, ...


build and run 
=============

[![Build Status](https://api.travis-ci.org/dbaeli/basic-webapp.svg?branch=master)](https://api.travis-ci.org/dbaeli/basic-webapp.svg)

```
mvn clean install

mvn tomcat7:run
```

And open:

   http://localhost:8088/basic-webapp
